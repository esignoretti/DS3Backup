Output: `internal/api/` package with HTTP server, router, handlers, types, and tests. No external HTTP framework — use Go standard library `net/http` with `net/http/httptest` for testing.

      - `POST /api/v1/start` — Start scheduler
      - `POST /api/v1/stop` — Stop scheduler
      
      **Job endpoints:**
      - `GET /api/v1/jobs` — List all jobs
      - `GET /api/v1/jobs/{id}` — Get job details — NOTE: Go 1.22+ supports path params via `{id}` pattern. Since this is Go 1.25, use `r.PathPrefix("/api/v1/jobs/")` and parse manually, OR use `http.ServeMux` pattern matching with `GET /api/v1/jobs/` and extract ID from path.
      
      **Backup endpoints:**
      - `POST /api/v1/backup/run/{id}` — Trigger a backup run for a specific job
      
      Add a simple logging middleware:
      ```go
      func loggingMiddleware(next http.Handler) http.Handler {
          return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
              log.Printf("API %s %s", r.Method, r.URL.Path)
              next.ServeHTTP(w, r)
          })
      }
      ```
    
    Create `internal/api/handlers.go`:
    
    - `(s *APIServer) handleStatus(w, r)` — GET /api/v1/status
      - Returns StatusResponse with scheduler running state, scheduled job count, API port, uptime
      - Uses RLock for thread safety
      - Status: 200
    
    - `(s *APIServer) handleStart(w, r)` — POST /api/v1/start
      - Calls `s.runner.Start()`
      - Returns `{ "status": "started" }`
      - Status: 200
    
    - `(s *APIServer) handleStop(w, r)` — POST /api/v1/stop
      - Calls `s.runner.Stop()`
      - Returns `{ "status": "stopped" }`
      - Status: 200
    
    - `(s *APIServer) handleListJobs(w, r)` — GET /api/v1/jobs
      - Calls `s.jobManager.GetAllJobs()`
      - Returns sanitized job list (remove EncryptionPassword field)
      - Status: 200
    
    - `(s *APIServer) handleGetJob(w, r)` — GET /api/v1/jobs/{id}
      - Extracts job ID from URL path
      - Calls `s.jobManager.GetJob(jobID)`
      - If nil: 404 error response
      - Returns sanitized job detail with schedule info
      - Status: 200
    
    - `(s *APIServer) handleRunBackup(w, r)` — POST /api/v1/backup/run/{id}
      - Extracts job ID from URL path
      - Validates job exists via `s.jobManager.GetJob(jobID)`
      - If nil: 404 error response
      - Calls `s.runner.RunJob(jobID)` (async — returns immediately)
      - Returns BackupTriggerResponse with triggered=true
      - Status: 202 (Accepted)
    
    - URL path parameter extraction helper:
      ```go
      func extractPathParam(path, prefix string) string {
          return strings.TrimPrefix(path, prefix)
      }
      ```
    
    Follow CONVENTIONS.md for function naming, error handling, JSON field conventions.
