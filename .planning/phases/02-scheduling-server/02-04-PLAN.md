    Add auto-start methods to `DaemonConfig`:
    
    ```go
    // InstallAutoStart creates a LaunchAgent plist on macOS (or equivalent on other platforms)
    func (d *DaemonConfig) InstallAutoStart(binaryPath string) error {
        switch runtime.GOOS {
        case "darwin":
            return d.installLaunchAgent(binaryPath)
        case "linux":
            return d.installSystemdService(binaryPath)
        default:
            return fmt.Errorf("auto-start not supported on %s", runtime.GOOS)
        }
    }
    
    func (d *DaemonConfig) RemoveAutoStart() error {
        switch runtime.GOOS {
        case "darwin":
            return d.removeLaunchAgent()
        case "linux":
            return d.removeSystemdService()
        default:
            return nil
        }
    }
    
    func (d *DaemonConfig) IsAutoStartInstalled() bool {
        // Check if plist/systemd file exists
    }
    ```

| T-02-11 | S | go mod tidy removing direct imports | mitigate | Verify `go build ./...` after tidy — build errors prevent silent dependency removal |
| T-02-12 | E | launchctl load subprocess | accept | Fails gracefully if launchctl unavailable — auto-start is best-effort |
</threat_model>
