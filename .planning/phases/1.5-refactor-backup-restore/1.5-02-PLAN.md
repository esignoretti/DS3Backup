    ```go
    // SetLifecyclePolicy configures S3 lifecycle policy for retention cleanup
    func (c *Client) SetLifecyclePolicy(ctx context.Context, retentionDays int) error {
        // Build lifecycle rule: expire objects in the backups/ prefix
        rule := s3.LifecycleRule{
            ID:     aws.String("ds3backup-retention"),
            Status: s3.ExpirationStatusEnabled,
            Filter: &s3.LifecycleRuleFilter{
                Prefix: aws.String("backups/"),
            },
            Expiration: &s3.LifecycleExpiration{
                Days: aws.Int32(int32(retentionDays)),
            },
            NoncurrentVersionExpiration: &s3.NoncurrentVersionExpiration{
                NoncurrentDays: aws.Int32(int32(retentionDays)),
            },
        }

        input := &s3.PutBucketLifecycleConfigurationInput{
            Bucket: aws.String(c.bucket),
            LifecycleConfiguration: &s3.BucketLifecycleConfiguration{
                Rules: []s3.LifecycleRule{rule},
            },
        }

        _, err := c.client.PutBucketLifecycleConfiguration(ctx, input)
        return err
    }
    ```

    **Replace `GetLifecyclePolicy`:**
    The current stub returns a static message. Replace with:

    ```go
    // GetLifecyclePolicy gets the current lifecycle policy
    func (c *Client) GetLifecyclePolicy(ctx context.Context) (string, error) {
        input := &s3.GetBucketLifecycleConfigurationInput{
            Bucket: aws.String(c.bucket),
        }

        output, err := c.client.GetBucketLifecycleConfiguration(ctx, input)
        if err != nil {
            // Check if no lifecycle config exists
            var noLifecycle *types.NoSuchLifecycleConfiguration
            if errors.As(err, &noLifecycle) {
                return "", nil
            }
            return "", fmt.Errorf("failed to get lifecycle policy: %w", err)
        }

        // Pretty-print the lifecycle configuration as JSON
        data, err := json.MarshalIndent(output, "", "  ")
        if err != nil {
            return "", fmt.Errorf("failed to marshal lifecycle config: %w", err)
        }

        return string(data), nil
    }
    ```

    Per D-03: Replace the no-op `applyRetention()` implementation with real deletion logic.

    The current `applyRetention` (lines 394-411) gets backup history, finds runs before the cutoff, and logs that they're "marked for deletion" but never deletes anything.

    **Required changes:**

    1. **Change the method signature** to accept a context and return an error:
       ```go
       func (e *BackupEngine) applyRetention(ctx context.Context, job *models.BackupJob) error {
       ```

    2. **Implement actual deletion logic:**
       - Get entries for each expired run using `e.indexDB.GetEntriesForRun()`
       - For each file entry, construct the S3 key (`backups/<jobID>/files/<hash>.enc` or batch key)
       - Call `e.s3client.DeleteObject(ctx, s3Key, true)` with `bypassGovernance=true` for GOVERNANCE mode
       - For COMPLIANCE mode: log a warning that the object cannot be deleted within retention period
       - Delete batch archives and manifests too
       - Track counts of deleted objects and failures

    3. **Update the call site** in `RunBackup` (line 247):
       Change from:
       ```go
       e.applyRetention(job)
       ```
       To:
       ```go
       ctx := context.Background()
       if err := e.applyRetention(ctx, job); err != nil {
           log.Printf("WARNING: Retention policy failed: %v", err)
       }
       ```

    **Note:** The `GetEntriesForRun()` method exists on the index but requires the exact backup time with 1-second tolerance. An alternative approach:
    - After getting the backup history, iterate runs before the cutoff
    - For each expired run, log a warning that S3 lifecycle will handle cleanup
    - Then ALSO scan S3 objects directly with `ListObjects(ctx, "backups/<jobID>/")` and parse object metadata to find expired entries beyond the retention window

    **Simpler approach (recommended):**
    Since we now have a real `SetLifecyclePolicy` (Task 1), the `applyRetention` method should:
    - For GOVERNANCE mode expired runs: call `DeleteObject` with `BypassGovernanceRetention: true` for the run's batch manifests, individual files, and any index copies stored with the backup
    - For COMPLIANCE mode: log a warning that objects are protected by COMPLIANCE lock until the retention period expires
    - Leave broader cleanup to the S3 lifecycle policy (which the user can set via `s3 lifecycle-set` command)
    - Log the number of objects deleted/warned about

    Implementation:
    ```go
    func (e *BackupEngine) applyRetention(ctx context.Context, job *models.BackupJob) error {
        cutoff := time.Now().AddDate(0, 0, -job.RetentionDays)

        runs, err := e.indexDB.GetBackupHistory(job.ID, 1000)
        if err != nil {
            return fmt.Errorf("failed to get backup history: %w", err)
        }

        var deletedCount int
        for _, run := range runs {
            if !run.RunTime.Before(cutoff) {
                continue
            }

            log.Printf("Applying retention for backup from %s", run.RunTime.Format(time.RFC3339))

            if job.ObjectLockMode == "COMPLIANCE" {
                log.Printf("WARNING: Cannot delete COMPLIANCE-locked backup from %s within retention period", run.RunTime.Format(time.RFC3339))
                continue
            }

            // Get entries for this run
            entries, err := e.indexDB.GetEntriesForRun(job.ID, run.RunTime)
            if err != nil {
                log.Printf("WARNING: Failed to get entries for run %s: %v", run.RunTime.Format(time.RFC3339), err)
                continue
            }

            for _, entry := range entries {
                if entry.S3Key != "" {
                    if err := e.s3client.DeleteObject(ctx, entry.S3Key, true); err != nil {
                        log.Printf("WARNING: Failed to delete %s: %v", entry.S3Key, err)
                        continue
                    }
                    deletedCount++
                }
            }
        }

        if deletedCount > 0 {
            log.Printf("Retention: deleted %d expired objects", deletedCount)
        }
        return nil
    }
    ```

    **Important:** The `GetEntriesForRun()` method signature accepts `(jobID string, targetTime time.Time)`. Verify the exact signature in `internal/index/index.go`. If it differs, adapt accordingly.
